# Product of Array Except Self

## Problem

Given an integer array nums, return an array answer such that answer[i] is equal to the product of all elements of nums except nums[i].

The solution must be implemented without using division.

---

## Approach

My first thought was to calculate the product of all elements and divide it by the current element for each index. However, I later realized that the problem specifically restricts the use of division.

I then implemented a brute force solution where, for every index, I traversed the entire array again and multiplied all elements except the current one. Although this worked, the time complexity was O(n²) due to the nested loops.

To optimize the solution, I used the prefix and suffix product technique.

I created two arrays:

* left[i] stores the product of all elements to the left of index i.
* right[i] stores the product of all elements to the right of index i.

The left array is built from left to right, while the right array is built from right to left.

Since the required answer for every index is simply:

Product of elements on the left × Product of elements on the right

the final result can be generated by multiplying the corresponding values from both arrays.

---

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)

---

## Key Learning

This problem demonstrated how precomputing information can eliminate repeated calculations. Instead of recomputing products for every index, prefix and suffix products allow each answer to be obtained in constant time after preprocessing.
